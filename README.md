# My slider

To use the slider you need to <a href="https://github.com/IlyaKavko/Slider">download</a> two files and copy them to your file

Add HTML markup to your `<body>`:

`<html>
    <div class="custom_slider">
        <div><img src="..." alt="..."></div>
        <div><img src="..." alt="..."></div>
        <div><img src="..." alt="..."></div>
    </div>
</html>`
